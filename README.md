# Seeker

AI agent that searches for relevant posts and responds.

Supports:

* Facebook


## Install

pip install -r requirements.txt

Copy and rename .env.example and setup as required.


## Develop

python3 -m venv .venv
source .venv/bin/activate
python main.py
